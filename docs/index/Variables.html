private $allow_redirect
Whether or not you want to be redirected when a Location header is retrieved.
private $groups
Registered groups, permission hook, in other words.
private $hooked
An array containing callbacks that have hooked themselves through the API...
private $hooked_actions
Another array containing actions plugins have registered (allows ?action=REGISTERED_ACTION)
private $hooked_subactions
Yup, thats right, an array containing sub-actions plugins can register on actions Please note that sub-actions only work if the plugin that registered the action uses this sub-action API
private $http_version
The HTTP version to use in requests, either 1.0 or 1.1
private $include_header
Whether or not you want to have the header included with the body of the retrieved file.
protected $js_files
Contains JavaScript files to include in the documents head tag.
protected $js_vars
An associative array (variable => value) containing JavaScript variables to be defined before including JavaScript files, if any.
protected $links
An array containing all the link tags to be added inside the head of the document.
protected $main_title
Contains the sites main title, this title is not independent per page but dependent on the whole sites name itself.
private $max_redirects
Specifies the maximum amount of times the Location header will be allowed to redirect your request, if allow_redirect is TRUE.
protected $meta
An array containing meta data to include in the documents head tag.
private $port
The port to connect through.
private $post_data
An array (key => value) containing POST data you want sent on every remote request.
private $referer
The value of the HTTP Referer header, if none set, the header is not sent.
private $settings
Contains the most up-to-date settings and values.
private $timeout
The maximum number of seconds that the downloading can take.
protected $title
The title specific to the current page.
private $update_settings
Contains the setting variables and values which are to be updated before the Settings object is destructed.
private $user_agent
The user agent to set the User-Agent HTTP header to.
