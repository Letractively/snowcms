Clean Request

Simply cleans the GLOBALS array in PHP of unwanted variables, such as old and deprecated variables (such as HTTP_GET_VARS), but it also removes what PHP’s magic quotes http://www.php.net/manual/en/security.magicquotes.php did to variables like $_GET, $_POST, $_REQUEST, etc.
