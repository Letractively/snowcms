Variables

main_title

protected $main_title

Contains the sites main title, this title is not independent per page but dependent on the whole sites name itself.

title

protected $title

The title specific to the current page.

url

protected $url

The URL of the current themes base directory.

links

protected $links

An array containing all the <link> tags to be added inside the <head> of the document.  Each array inside links can contain charset, href, hreflang, media, rel, rev, target, type, class, dir, id, lang, style, title and xml:lang.

js_vars

protected $js_vars

An associative array (variable => value) containing JavaScript variables to be defined before including JavaScript files, if any.

js_files

protected $js_files

Contains JavaScript files to include in the documents <head> tag.  Each array inside can contain type, charset, defer and src.

meta

protected $meta

An array containing meta data to include in the documents <head> tag.  Each array can contain content, http-equiv, name, scheme, dir, lang and xml:lang.

return_js_var

public function return_js_var($variable =  null)

Returns the specified JavaScript variables value, or all of them.

Parameters

string $variableThe name of the variable to return, if set to null all JavaScript variables will be returned.

Returns

mixedReturns the set value for the specified variable, if $variable is set (false if it does not exist), if $variable is not set, then an array containing all the current variables is returned.

Note

Even though false is returned if the a variable is specified does not necessarily mean it is not set, since the variables value can also be set to false.  In which case, you should check with Theme::js_var_exists.
