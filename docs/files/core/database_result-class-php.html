Database_Result

This is another abstract class for SnowCMS’s databasing system.  This is a class which is returned by the Database::query and Database::insert methods.

free_result

abstract public function free_result()

Frees all memory associated with the result received from a query execution.

Parameters

none

Returns

boolReturns TRUE on success, FALSE on failure.

Note: According to most people, you should ONLY free a result IF and ONLY IF the query you executed returns huge result sets, otherwise you can end up using more memory in the process of freeing the result.
